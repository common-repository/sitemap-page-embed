Sitemap Page Embed plugin adds the sitemap of your site anywhere on a page through tinymce button or using a simple shortcode [spe_sitemap_shortcode]. 

== Description ==

What does it do

*   Embeds your sitemap pages list on your page in human readable format.
*   The sitemap is not in XML format because it is for humans not for search engine crawlers.

How to Use and Install

	* Activate plugin
	* Go to the edit screen of a page 
	* Go to visual tab
	* Press the golden button on the tinymce to add the pages sitemap of your site.

About

	The Sitemap Page Embed plugin is maintained by http://wordpresshowtos.blogspot.com/2015/10/sitemap-page-embed.html
